# ChemCompute

ChemCompute is a Python and Streamlit-based digital chemistry assistant designed for students, educators, and researchers.

## Features
- Element Search
- Molar Mass Calculator
- Bond Prediction
- Reaction Analysis

## Technologies Used
- Python
- Streamlit

## Run Locally

```bash
streamlit run app.py
```

## Project Goal
To simplify chemistry calculations and support digital learning environments.
